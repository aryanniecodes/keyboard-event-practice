# Keyboard Event Visualizer

This is a simple web project built using HTML, CSS, and JavaScript to practice keyboard events.

It displays the key pressed by the user in real time using the `keydown` event and updates the content dynamically on the screen.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

Open the `index.html` file in any web browser.

## Educational Purpose

This project is created for educational purposes to understand JavaScript keyboard events and basic DOM manipulation.
